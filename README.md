# Sarthak-s-Digital-Literacy-Project
📖 Overview

This project is part of the Digital Literacy Course (CSE0001).
The objective of this project is to demonstrate practical knowledge of digital tools, online collaboration, and safe internet practices.

🎯 Objectives

To develop skills in using modern digital tools
To create and manage an online presence
To understand professional communication and email etiquette
To learn about cybersecurity and safe online behavior

🛠️ Tools & Technologies Used

Canva (for design and presentations)
GitHub (for version control and project hosting)
Google Docs / Slides / Sheets (for collaboration)
LinkedIn (for professional profile building)

📂 Project Description

In this project, I have:

Created digital content such as presentations and/or designs
Built and maintained a GitHub repository
Explored online platforms for professional and academic growth
Applied email and communication best practices
Studied cybersecurity awareness and prevention techniques

🚀 Features / Work Done

Designed visuals using digital tools
Organized project files using GitHub
Documented work clearly and professionally
Applied digital communication standards
Demonstrated safe and responsible internet usage

🔐 Cybersecurity Awareness

This project also highlights:

Common cyber threats (phishing, scams, etc.)
Safe password practices
Importance of protecting personal data

📊 Learning Outcomes

Through this project, I learned:

Effective use of digital tools for productivity
How to collaborate online
Importance of digital identity and portfolio
Basics of cybersecurity and digital safety
